# 貢献者行動規範（Contributor Covenant Code of Conduct）

🇬🇧 English: **[CODE_OF_CONDUCT.md](./CODE_OF_CONDUCT.md)**（こちらがメインです）

## 私たちの誓い

メンバー、コントリビューター、およびリーダーである私たちは、年齢、体の大きさ、目に見える・
見えない障がい、民族性、性的特徴、性自認と性表現、経験の度合い、教育、社会経済的地位、国籍、
容姿、人種、宗教、性的指向に関わらず、私たちのコミュニティへの参加が誰にとってもハラスメントの
ない体験となるようにすることを誓います。

私たちは、オープンで歓迎的、多様で包摂的、そして健全なコミュニティに貢献する形で行動し、交流する
ことを誓います。

## 私たちの行動基準

コミュニティにとって前向きな環境に貢献する行動の例：

- 他者への共感と優しさを示すこと
- 異なる意見・視点・経験を尊重すること
- 建設的なフィードバックを与え、また快く受け入れること
- 自分の過ちによって影響を受けた人々に対して責任を受け入れ、謝罪し、その経験から学ぶこと
- 個人としてだけでなく、コミュニティ全体にとって最善であることに焦点を当てること

容認できない行動の例：

- 性的な言葉や画像の使用、およびあらゆる種類の性的な注目や誘い
- 荒らし行為、侮辱的・軽蔑的なコメント、個人的・政治的な攻撃
- 公的または私的なハラスメント
- 明示的な許可なく、物理的住所やメールアドレスなど他者の私的情報を公開すること
- 職業上の場において不適切と合理的に判断されるその他の行為

## 執行の責任

コミュニティのリーダーは、容認できる行動の基準を明確にし、執行する責任を負い、不適切・脅迫的・
攻撃的・有害とみなす行動に対して、適切かつ公正な是正措置を講じます。

コミュニティのリーダーは、本行動規範に沿わないコメント、コミット、コード、Wiki の編集、Issue、
その他の貢献を削除・編集・拒否する権利と責任を持ち、適切な場合にはモデレーションの理由を説明
します。

## 適用範囲

本行動規範は、コミュニティのすべての空間に適用され、また個人がコミュニティを公の場で公式に代表
している場合にも適用されます。コミュニティを代表する例としては、公式のメールアドレスの使用、公式
ソーシャルメディアアカウントでの投稿、オンライン・オフラインのイベントでの指名された代表としての
行動などが挙げられます。

## 執行

虐待的・ハラスメント的・その他容認できない行動の事案は、リポジトリの **Security** タブからの非公開
報告、または GitHub 上の **@Y1-Effy** への連絡により、プロジェクトのメンテナーに非公開で報告でき
ます。すべての苦情は迅速かつ公正に検討・調査されます。

すべてのコミュニティリーダーは、いかなる事案についても報告者のプライバシーとセキュリティを尊重する
義務を負います。

## 執行ガイドライン

コミュニティのリーダーは、本行動規範に違反するとみなした行動の結果を判断する際、以下の
コミュニティ影響度ガイドラインに従います。

### 1. 訂正

**コミュニティへの影響**: 不適切な言葉づかい、またはコミュニティにおいて非専門的・歓迎されないと
みなされるその他の行動。

**結果**: コミュニティリーダーからの非公開かつ書面での警告。違反の性質を明確にし、その行動がなぜ
不適切であったかを説明します。公開の謝罪が求められる場合があります。

### 2. 警告

**コミュニティへの影響**: 単一の事案、または一連の行動による違反。

**結果**: 継続した場合の結果を伴う警告。一定期間、関係者との交流（行動規範を執行する者への一方的な
接触を含む）を行わないこと。これにはコミュニティ空間内だけでなく、ソーシャルメディアなどの外部
チャネルでの交流の回避も含まれます。これらの条件に違反した場合、一時的または恒久的な追放につながる
ことがあります。

### 3. 一時的な追放

**コミュニティへの影響**: 持続的な不適切行動を含む、コミュニティ基準の重大な違反。

**結果**: 一定期間、コミュニティとのあらゆる種類の交流や公的なコミュニケーションからの一時的な
追放。この期間中、関係者との公的・私的な交流（行動規範を執行する者への一方的な接触を含む）は
許可されません。これらの条件に違反した場合、恒久的な追放につながることがあります。

### 4. 恒久的な追放

**コミュニティへの影響**: 持続的な不適切行動、個人へのハラスメント、特定の集団に対する攻撃や
軽視など、コミュニティ基準の違反パターンを示すこと。

**結果**: コミュニティ内でのあらゆる公的な交流からの恒久的な追放。
